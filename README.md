# angular-sbt
Teste de conhecimento em Angular, para vaga no SBT

Desafio consiste em: 
- Apresentação de Case do Candidato(escolha livre)
- Em Angular
- Utilização de rotas
- Aplicação de modularização
- Ajax (com tratamento e exibição de retorno)
- Utilização de HTML5
- CSS3

Foi ultilizado json-serve para persistência de dados, portanto é importante startar o mesmo, senão os dados não virão e não serão postados.

## comandos
### "npm start"
### "npm run json"
